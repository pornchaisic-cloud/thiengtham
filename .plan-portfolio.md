# Plan: Portfolio - Video + Centering

## Problem
1. Reel (video) posts show as static thumbnail, video doesn't play
2. Single image/video is left-aligned instead of centered

## Changes

### 1. Data structure
`{ src: string; url: string }[]` → `{ src: string; url: string; isVideo: boolean; videoSrc?: string }[]`

### 2. Fetch `media.source` (video URL)
- If `att.media.source` exists → `isVideo: true`, `videoSrc` = video URL
- If only `att.media.image.src` → `isVideo: false`

### 3. Increase `limit=5` for more posts
`&limit=1` → `&limit=5`

### 4. Template
```astro
{#if item.isVideo}
  <video poster={item.src} controls muted playsinline>
    <source src={item.videoSrc} type="video/mp4">
  </video>
{:else}
  <img src={item.src} ...>
{/if}
```

### 5. CSS centering
Replace `columns: 3` with grid/flex:
- `≥2 items`: grid 3 cols
- `--single`: flexbox centered
